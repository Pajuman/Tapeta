Tapeta

Tato webová aplikace umožňuje zakoupit tapetu s uživatelem vloženým textem. Uživatel má na výběr z několika grafických předloh (tapet). Uživatel si do inputu vepíše text, který se pak zobrazí na tapetě. Tiskový výstup je v PDF. Služba je placená.

Hlavní komponenty:
- UI umožňující výběr tapety, fontu a psaní textu
- renderování tapety + textu do pdf
- platbu

Jak si to představuju (uživatelský pohled)
Uživatel přijde na stránku a:

Vybere si jeden z návrhů polepu
bude tam cca 5 variant (různé barvy/styly)- jeden vkládám do přílohy 
všechny mají nad dveřmi ceduli (prázdné místo pro text)
Vybere si font
cca 3 fonty
já dodám
Napíše vlastní text
ten se mu hned zobrazí v obrázku na ceduli
Uvidí náhled
jak to bude vypadat finálně
Potvrdí, že je to OK
Zaplatí
Stáhne si PDF
připravené pro tisk
